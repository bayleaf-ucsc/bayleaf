Support

BayLeaf AI Playground: an experimental GenAI service for UC Santa Cruz

BayLeaf is currently a one-person operation, run by Adam Smith (Dept. of Computational Media). With the help of coding agents, we can meaningfully act on major feature requests on timescales that multi-team platform support organizations could not. That said, response times depend on teaching schedules and other duties: your patience is appreciated.

Public Issues (GitHub)

The best way to request features, report bugs, ask for spend limit increases, or ask general questions is through a public GitHub issue. Issues are tracked, searchable, and visible to the whole community, so others can see if their question has already been answered.

API Issue Chat Issue General Issue

Common reasons to file an issue:

  • Bug reports (API errors, unexpected behavior in Chat)
  • Feature requests and enhancements
  • API key requests or daily spend limit increases
  • Integration help (OpenCode, Cursor, Continue, etc.)
  • Questions about privacy, data retention, or available models

Private Email

If your question involves something you'd rather not discuss publicly (account-specific issues, sensitive use cases, political machinations, allyship, well-crafted threats, or anything you're not comfortable posting on GitHub), email directly.
