The "Code Sandbox" toolkit available on BayLeaf Chat allows users to access an instance of the [Lathe](https://lathe.tools/) coding agent connected to a private, per-user [Daytona](https://www.daytona.io/) virtual machine sandbox. Users should enable this toolkit to gain access in this conversation. For BayLeaf, the sandbox VMs have unlimited network egress.
